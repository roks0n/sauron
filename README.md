# Sauron

```bash
HTTP service health watcher

USAGE:
    sauron <SUBCOMMAND>

OPTIONS:
    -h, --help    Print help information

SUBCOMMANDS:
    add       Add thing
    help      Print this message or the help of the given subcommand(s)
    list      List things
    remove    Remove thing
    run       Run a one-off check
    setup     
    watch     Watch
```
